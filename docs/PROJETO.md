# Projeto do Site da IPI Pirapozinho

## Nome

Site Institucional da IPI Pirapozinho

## Objetivo

Criar uma página estática simples, bonita e responsiva para a Igreja Presbiteriana Independente de Pirapozinho-SP.

## Escopo inicial

- apresentação da igreja
- programação semanal
- endereço
- WhatsApp oficial
- redes sociais
- links úteis da IPIB

## Fora do escopo inicial

- boletim completo
- avisos muito temporários
- conteúdo que muda toda semana sem necessidade

## Princípio

O site precisa ser leve, fácil de manter e com informação essencial.
